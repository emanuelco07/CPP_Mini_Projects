In folderul prezent avem urmatoarele proiecte codate in c++:

CoinFlipSimulator - o mini aplicatie ce ne permite sa simulam aruncarea cu banul
CountDownTimer - o aplicatie ce este un temporizator
DiceSimulator - aplicatie ce ne permite sa simulam aruncarea cu zarul
GuessingNumber - o aplicatie ce simuleaza ghicirea unui numar
LoterryNumber - mini aplicatie ce simuleaza loteria romana
RockPaperScissor - o simulare a jocului piatra hartie foarfeca
SavingsCalculator - aplicatie ce ne calculeaza banii pe care ii castigam in functie de banii depusi intr un depozit, dobanda anuala si suma pe care o adaugam anual in cont
Hangman - joc de spanzuratoarea
Linia: "system("cls")" este folosita pentru windows, daca aveti mac, folositi comanda "system("clear")".
